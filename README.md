## overview 🌍
this project simulates the movement of a drunk pedestrian in a 2d world using a client-server architecture. 🚶‍♂️💨

## requirements 📋
- c99 or higher. 📖
- a compatible compiler (gcc). ⚙️
- basic knowledge of socket programming. 💻
